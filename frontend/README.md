# Instalação

É necessário instalar as dependências e bibliotecas necessárias para rodar a aplicação.

Para isto, esteja ciente que está dentro da pasta _frontend_ para executar o seguinte comando:
```
npm i
```

Após a instalação dos pacotes, se for necessário, modifique a url da API no arquivo **.env** para o desejado.

Feito esses passos, será possível utilizar a aplicação.
